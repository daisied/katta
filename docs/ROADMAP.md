# Roadmap

## v1.0 (Current Foundation)

- GitHub-ready repository hygiene
- Runtime bootstrap templates
- CI + tests + lint gates
- Security and contribution docs

## v1.1 (Product WOW)

- Slash-command UX (`/research`, `/sources`, `/status`)
- Rich response cards with source confidence markers
- Optional web dashboard for runtime health and queue visibility

## v1.2 (Research Quality WOW)

- Deterministic benchmark harness with scoring reports
- Source diversity tracking (avoid single-platform bias)
- Regression threshold checks before release tags

## v1.3 (Safety and Ops WOW)

- Fine-grained policy profiles per channel/user
- Optional sandboxed command executor sidecar
- Observability package (metrics + structured traces)
